# 实用函数库

## Ramda

[Ramda](https://ramdajs.com/)

> Ramda 函数会自动柯里化。
> Ramda 函数的参数经过排列，以便于柯里化。要操作的数据通常是最后提供的。
> 两点使得将函数构建为更简单函数的序列变得非常容易，每个函数都会转换数据并将其传递到下一个函数。 Ramda 旨在支持这种编码风格。

和 lodash 的最大区别在于操作的数据放在最后
