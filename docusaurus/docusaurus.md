# docusaurus

[Home](https://docusaurus.io/)

文档创建工作，facebook 出品。

可使用 React 组件，或是 Markdown。

## pages

src/pages/ 下

自动为 src/pages/ 内的所有 JavaScript/TypeScript 文件生成相应的网站路径。

可以使用 Markdown,可以使用 MDX

## docs

docs/ 下

默认根据目录结构生成侧边栏，也可显性定义

可以分版本，但是一般不需要
