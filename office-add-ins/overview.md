## 使用 Script Lab

一个沙箱环境

### 安装

- 在 Insert -> Store 里安装，可能要翻墙
- 等安装完成后，在 Insert -> My Add-ins -> See All 里添加
- 工具栏上的 Script Lab 页使用

如果打开 Script Lab 白屏，可能要翻墙

没有找到太多文档，直接去 excel 里看 samples，左上角图标 -> Samples
